Projects

I worked on a Wikipedia Vandalism Detection project in a group from Sep-Nov 2016. The goal was to build a model that can detect a fraudulent or incorrect revision based on revision data from WikiDumps using scikit-learn and pandas. I initially trained a logistic regression and helped clean the data. I later used dimensionality reduction techniques and researched which features were the most significant to improve the accuracy of our final random forest model.
